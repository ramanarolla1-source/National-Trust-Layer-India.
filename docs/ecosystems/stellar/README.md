# NTL x Stellar: Agentic Compliance Infrastructure
### *Stellar Hacks: Agents Submission*

## 🌐 Vision
Providing "Institutional Guardrails" for the emerging AI agent economy. We use Stellar’s native payment rails to move from unverified autonomous actions to **Accountable Agentic Commerce**.

## 🛠 Technical Integration
- **Payment Protocol:** [x402](https://stellar.org/x402) (HTTP-native micropayments).
- **Settlement:** Stellar Asset Sandbox (USDC/XLM).
- **Smart Contracts:** [Soroban](https://stellar.org/soroban) for spending policies and mandate verification.
- **Agent Framework:** ElizaOS hosted on Nosana GPU nodes.

## 📈 The "Sovereign Audit" Use Case
Most AI agents fail when they hit a "Payment Wall." NTL solves this via:
1. **Pay-per-Audit (x402):** Agents autonomously "buy" a compliance check from the NTL Audit Agent before settling a high-value trade.
2. **Soroban Guardrails:** If a machine-to-machine (MPP) payment exceeds a user-defined "Mandate," the Soroban contract blocks the x402 facilitator until a hardware-anchored biometric signature is provided.
3. **Interoperability:** Aligning with **ISO 20022** standards to allow agents to interact with traditional financial messaging (SWIFT integration).
