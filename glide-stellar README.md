# Glide Stellar 🚀
### The Frictionless UX Gateway to the Stellar Network

![Glide Stellar Banner](./Glide%20Stellar.jpg)

## 📖 Overview
**Glide Stellar** is a high-performance UX middleware designed to bridge the gap between Stellar’s powerful infrastructure and the everyday user. While the Stellar network offers world-class speed and low costs, the "last mile" of user experience remains a barrier for mass adoption. 

Glide provides a streamlined interface layer that automates complex technical tasks—like **Trustline management**, **Pathfinding**, and **SEP-24 integrations**—into a single-click experience.

---

## ✨ Key Features
* **Zero-Friction Onboarding:** Integration of social logins and non-custodial wallet abstraction.
* **Smart Trustlines:** Automated detection and creation of trustlines for incoming assets.
* **Optimized Pathfinding:** Ensuring users always get the best rate for cross-asset swaps without needing to understand liquidity pools.
* **Institutional Readiness:** A UI designed for RWA (Real World Assets) and DePIN projects requiring high-fidelity dashboards.

---

## 🛠️ Technical Stack
* **Core:** Stellar TypeScript SDK
* **Smart Contracts:** Soroban (Rust)
* **Standards:** SEP-10 (Auth), SEP-24 (Deposits/Withdrawals), SEP-30 (Recovery)
* **Frontend:** React / Tailwind CSS
