# NTL x Starknet: Institutional Trust Infrastructure
### *Starknet Seed Grant Submission*

## 🌐 Vision
Utilizing Starknet's **Native Account Abstraction (AA)** to bridge the "Anonymity Debt" in institutional finance. We are moving from unverified social identities to **Hardware-Anchored Identity (HAI)**.

## 🛠 Technical Integration
- **SDK:** [Starkzap SDK v2](https://www.starkzap.com/)
- **Wallet Logic:** Cartridge Controller for Biometric Passkeys.
- **Security:** Silicon-to-Chain signatures via device Secure Enclaves.

## 📈 Innovation & Use Case
Current RWA (Real World Asset) projects struggle with the **March 2026 Eurosystem standards**. NTL solves this by:
1. **Universal Mandate Dashboard:** Allowing users to set BTC/Stablecoin spending limits directly on-chain.
2. **Institutional UX:** Replacing seed phrases with hardware-bound biometrics (FaceID/Fingerprint).
3. **Compliance:** Aligning with the **July 2026 Indian VDA mandates**.
